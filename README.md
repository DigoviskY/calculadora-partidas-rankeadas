# Calculadora de Partidas Rankeadas

Projeto desenvolvido como parte dos desafios da plataforma DIO, com foco em lógica de programação e JavaScript.

## Objetivo
Criar uma função que recebe vitórias e derrotas, calcula o saldo e define o nível do herói.

## Conceitos aplicados
- Variáveis
- Funções
- Estruturas condicionais
- Lógica de programação

## Regras de nível
- Menos de 10 vitórias → Ferro  
- 11 a 20 → Bronze  
- 21 a 50 → Prata  
- 51 a 80 → Ouro  
- 81 a 90 → Diamante  
- 91 a 100 → Lendário  
- 101 ou mais → Imortal  

## Exemplo

calcularNivel(75, 20);

Saída:
O Herói tem de saldo de 55 está no nível de Ouro
